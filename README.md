# README

This repository contains all materials for the Creative Computing System project.

## Notes and Documentation
All written documentation, reflections, and project notes are created in **Obsidian (Markdown) format**.  
They can be found at the following path:

System-for-Creative-Computing/notes/SFCC

## Practical Work and Code
All practical work files are organised into separate folders:

- **python files**  
  Contains all Python scripts related to the project, written in `.py` format.

- **PD**  
  Contains program files created using **Pure Data**, including patches used for experimentation and practice.

This structure separates documentation from practical work to make the project easier to review.
